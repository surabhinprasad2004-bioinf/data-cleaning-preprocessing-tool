# data-cleaning-preprocessing-tool

## Overview
This project automates data preprocessing tasks including:
- Handling missing values
- Removing duplicate records
- Detecting and removing outliers using the IQR method

## Libraries Used
- Pandas
- NumPy

## Files
- data.csv (raw dataset)
- data_cleaning.py (Python script)
- cleaned_data.csv (processed dataset)

## Output
A cleaned dataset suitable for data analysis and machine learning applications.
